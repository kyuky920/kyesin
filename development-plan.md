# 2026 계신 청년 하계수련회 웹 서비스 개발 기획안

## 1. 프로젝트 개요

- 프로젝트명: 2026 계신 청년 하계수련회 웹 서비스
- 브랜드 문구: WALK WITH HIM
- 주제: AI시대에도 하나님과 동행하기
- 말씀: 창세기 5:24
- 장소: 명륜교회
- 기간: 2026-07-30 ~ 2026-08-01

이 프로젝트는 계신 청년 하계수련회를 위한 전용 웹 서비스다. 참가자는 빠르게 자신의 조편성 결과와 수련회 일정을 확인할 수 있어야 하고, 운영진은 참석자 명단 관리와 조편성 운영을 효율적으로 수행할 수 있어야 한다.

## 2. 프로젝트 목표

### 참가자용 목표

- 본인 확인 후 자기 조를 빠르게 확인할 수 있어야 한다.
- 수련회 일정표와 장소 안내를 모바일에서 쉽게 볼 수 있어야 한다.
- 조편성 변경이 발생해도 최신 상태를 바로 확인할 수 있어야 한다.

### 관리자용 목표

- 참석자 명단을 업로드하고 정리할 수 있어야 한다.
- 기준에 맞는 자동 조편성을 실행할 수 있어야 한다.
- 운영 상황에 따라 조원을 수동으로 이동하거나 수정할 수 있어야 한다.
- 조별 현황, 인원 편차, 교회 중복, 성별 분포를 빠르게 검토할 수 있어야 한다.

## 3. 사용자 구분

### 1) 참가자

- 별도 회원가입 없음
- 본인 확인 방식:
  - 소속교회
  - 이름
  - 생년
- 모바일 중심 사용

### 2) 관리자

- 수련회 운영 담당자
- 데스크톱 우선 사용
- 참석자 관리, 조편성, 일정/장소/공지 운영 담당

## 4. 현재 기준 데이터 현황

현재 확보된 참석자 명단 기준 요약:

- 총 참석자: 136명
- 남성: 80명
- 여성: 56명
- 숙박 예정: 95명
- 비숙박: 41명
- 참석 패턴:
  - 3일 참석: 83명
  - 금/토 참석: 31명
  - 목/금 참석: 13명
- 조 규모 목표:
  - 기본 4~5명
  - 운영상 약 28~34개 조 범위 예상

교회명은 일부 표기가 혼재되어 있어 정규화가 필요하다.
예:
- 가락동부 / 가락동부교회
- 광흥 / 광흥교회
- 명륜 / 명륜교회
- 평강 / 평강교회 / 평촌 평강교회

## 6. 조편성 운영 기준

조편성은 다음 기준으로 수행한다.

1. 연령대별 우선 편성
   - 20세~24세
   - 25세~28세
   - 29세 이상
2. 성별은 최대한 균등하게 배치
3. 참여 일자를 고려했을 때 조별 실제 유효 인원이 최대한 균등하도록 배치
4. 같은 교회 인원은 최대한 같은 조에 겹치지 않도록 분산
5. 단, 초월제일교회 인원은 항상 같은 조가 되도록 고정
6. 자동 편성 이후에도 관리자가 수동으로 자유롭게 조정 가능해야 함

### 조편성 UX 원칙

- 자동 편성은 좋은 초안을 빠르게 만들어주는 역할
- 최종 운영 판단은 관리자가 수행
- 충돌 조건은 경고로 표시
- 수동 이동은 허용하되, 연령대 이탈/교회 중복/성별 편차 등을 즉시 보여줘야 함

## 8. 화면별 목적

### 홈

- 수련회 브랜딩과 핵심 진입점 제공
- 참가자용/관리자용 주요 진입 버튼 제공
- 주제, 말씀, 기간, 장소 표시
- 주요 일정 미리보기 제공

### 참가자 확인

- 소속교회, 이름, 생년으로 본인 확인
- 로그인 없는 간단한 진입 경험 제공

### 내 조 화면

- 본인 정보
- 배정 조 이름/번호
- 조원 목록
- 다음 일정 요약
- 장소 안내 진입

### 일정표 화면

- 날짜별 일정 목록
- 시간, 제목, 세부 설명, 강사, 장소 표시

### 장소 안내 화면

- 본당, 식당, 숙소, 조별나눔실, 외부활동 공간 안내

### 관리자 대시보드

- 전체 참석자 수
- 조 배정 완료 수
- 미배정 수
- 조 개수
- 연령대 분포
- 운영 바로가기

### 참석자 관리 화면

- 참석자 목록
- 교회, 연령대, 생년, 참석일수, 조 배정 여부 확인
- 이후 필터/검색/수정 기능 확장 예정

### 조편성 관리 화면

- 자동 편성 실행
- 조별 카드형 표시
- 조원 목록 표시
- 남/여 수, 교회 수, 참석일 분포 표시
- 이후 드래그 이동, 고정 조건 편집 기능 확장 예정

## 9. UX/UI 방향

### 전체 톤앤매너

- 딥네이비 기반의 차분하고 신뢰감 있는 분위기
- 교회 수련회 특유의 따뜻함과 현장성을 유지
- 너무 일반적인 SaaS 느낌보다 행사 전용 서비스처럼 설계

### 참가자용 UX 원칙

- 모바일 퍼스트
- 3초 안에 내 조를 찾는 경험
- 정보 접근 단계를 최소화
- 중요한 정보만 먼저 노출

### 관리자용 UX 원칙

- 데스크톱 우선
- 조편성과 운영 현황을 한눈에 확인 가능
- 자동화와 수동 조정의 균형
- 반복 수정에 강한 구조
